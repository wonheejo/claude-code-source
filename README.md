# Claude Code Source

Archived source code from [Claude Code](https://claude.ai/code), Anthropic's official CLI tool for Claude.

## Overview

This repository contains the `src/` directory extracted from the `@anthropic-ai/claude-code` npm package. It is a TypeScript/React (Ink) CLI application built with Bun.

## Structure

Key directories:

- `src/commands/` — CLI command definitions
- `src/tools/` — Tool implementations (Read, Write, Edit, Bash, Grep, etc.)
- `src/skills/` — Skill system (commit, review-pr, etc.)
- `src/components/` — Ink (terminal UI) React components
- `src/services/` — Core services (auth, API, config)
- `src/hooks/` — Lifecycle hooks
- `src/server/` — Local server functionality
- `src/voice/` — Voice input support
- `src/vim/` — Vim mode support
- `src/keybindings/` — Keybinding configuration
